# System Requirements

## Overview

The application is a web-based investment platform that enables users to monitor financial instruments, manage investment portfolios, and execute simulated trading operations. The system also supports automated portfolio management based on predefined risk profiles.

## Functional Requirements

### User Management

* User registration.
* User authentication (login).
* User logout.

### Market Data Visualization

* Display price charts for selected financial instruments.
* Display technical analysis indicators.
* Change the chart time interval.
* Change the displayed financial instrument.

### Watchlist Management

* Add financial instruments to a watchlist.
* Remove financial instruments from a watchlist.
* View the watchlist.

### Portfolio Management

* Buy financial instruments.
* Sell financial instruments.
* View transaction history.
* View currently held assets.
* View an investment portfolio summary.

### Automated Portfolio Management

* Enable automated portfolio management based on a selected risk profile.

## Non-Functional Requirements

### Security

* The system shall provide authentication and authorization mechanisms to protect user data and resources.

### Scalability

* The system shall support multiple concurrent users without significant performance degradation.

### Maintainability

* The system architecture shall be modular and extensible, allowing future development and modifications.

### Usability

* User interface elements shall be organized in a logical and intuitive manner, minimizing the number of interactions required to perform common tasks.
